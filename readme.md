## Usage

NODE_ENV = development
PORT = 5000
MONGO_URI = mongodb+srv://vihan:vihan@cluster0.r62xhrp.mongodb.net/databasename?retryWrites=true&w=majority
JWT_SECRET = abc123

### Install dependencies

```
# Backend deps
npm install

# Frontend deps
cd frontend
npm install
```

### Run Server

```
npm run server
```
